## chapter 1 逻辑命题
**原子命题** 
命题逻辑中最基本的单位
- 无逻辑连接词
- 不可拆分
- 确定真假

**逻辑连接词**
- 否定
- 合取（且）：同真为真
- 析取（并）：同假为假
- 蕴含：前真后假为假，其余为真
- 等价：同真同假为真
- 异或：一真一假为真

**逻辑符号可交换性**
二元 并且 非蕴含关系 
- 合取
- 析取
- 等价
- 异或

**命题公式按照逻辑分类**
- **重言式（永真式）**：真值恒为 1
- **可满足式**：至少有一种赋值使其为真
- **矛盾式（永假式）**：真值恒为 0

``` 
否定相关
1. 双重否定律
    ¬¬p≡p
2. 德摩根律
    ¬(p∧q)≡¬p∨¬q  
    ¬(p∨q)≡¬p∧¬q

真值常量相关
3. 矛盾律
    p∧¬p≡0
4. 排中律
    p∨¬p≡1
5. 零律
    p∧0≡0 
    p∨1≡1
6. 同一律
    p∧1≡p
	p∨0≡p

幂等、交换、结合律

7. 幂等律
    p∨p≡p
	p∧p≡p
8. 交换律
    p∨q≡q∨p
	p∧q≡q∧p
9. 结合律
    
    (p∨q)∨r≡p∨(q∨r)
	(p∧q)∧r≡p∧(q∧r)

分配与吸收律

10. 分配律
    
    p∧(q∨r)≡(p∧q)∨(p∧r)
	p∨(q∧r)≡(p∨q)∧(p∨r)
11. 吸收律
    
    p∨(p∧q)≡p
	p∧(p∨q)≡p

蕴含与等价相关

12. 蕴含等值式
    p→q≡¬p∨q
13. 等价等值式
    p↔q≡(p→q)∧(q→p)
	p↔q≡(¬p∨q)∧(¬q∨p)
	
14. 假言易位（逆否命题）
    p→q≡¬q→¬p
15. 等价否定等值式
    p↔q≡¬p↔¬q
16. 归谬论
    (p→q)∧(p→¬q)≡¬p
```

**什么是命题**
此时不考虑真假
- 陈述句
- 有确定的真假
	- 自指悖论，无法确定真假
	- 含有未知数，无法确定真假
- 不能是主观臆断


``` 

易混淆例子
“明天会下雨。”
“你现在在想什么？”
“请回答这个问题。”
“1/0 = 0。”
“本语句是假的。”
“这道菜太咸了。”
“宇宙是无限的。”

特殊注意 
1、数学中的命题：通常指可以证明真假的陈述。 
2、悖论：像“我在说谎”这种自指矛盾句不是命题。 
3、未来偶然事件：如“明天下雨”在逻辑学中通常视为命题（二值原则），但在某些哲学或多值逻辑中可能有争议。经典逻辑中算命题。 
4、命题通常用大写字母 P、Q、R 等表示。
```

## chapter 2 谓词逻辑

**基础概念**
- **个体**：所研究对象中的具体或抽象事物。
- 个体常元：a,b,c,…（表示特定对象）
- 个体变元：x,y,z,…（表示任意对象）

- **谓词**：表示个体性质或者个体之间的关系
	- 一元谓词P(x)：表示性质
	- 二元谓词Q（x，y）：表示关系
	- n元谓词：R（x1，x2，···)

- **量词**
	- 全称量词 ∀x
	- 存在量词 ∃x

- **谓词公式的组成**
	- 原子公式（谓词符号+项）
	- 连接词（¬, ∧, ∨, →, ↔）
	- 量词（∀, ∃）
	- 括号

- **命题函数**
	含自由变元的谓词表达式
	- 赋值后才是命题

- **语句**
	不含自由变元的谓词公式

- **个体域（论域）**：个体变项的取值范围。

**量词辖域扩张/收缩**（当x不在Q中自由出现时）

- ∀x(P(x)∧Q)≡∀xP(x)∧Q

- ∃x(P(x)∧Q)≡∃xP(x)∧Q

**量词次序**
``` 
量词次序
∀x∀yP(x,y)≡∀y∀xP(x,y）
∃x∃yP(x,y)≡∃y∃xP(x,y)    
不可随意交换：∀x∃yP(x,y)≢∃y∀xP(x,y)
```


**符号化**

多重量词复杂的符号化需要练一下，不然要混

- 全称量词通常与 → 搭配
- 存在量词通常与 ∧ 搭配

```md
1. 确定论域（个体域）：讨论对象的集合。
2. 识别个体常元：特定对象的符号（如a: 张三）。
3. 选择谓词符号：表示性质或关系（如H(x): x是人）。
4. 分析量词：找出“所有”“存在”等词语。
5. 用联结词组合：注意“且”“或”“如果…则”等。
```

## chapter 3 集合

- **含于** ：集合之间的关系
- **属于**：元素与集合之间的关系

- 关于**空集**作为元素和空集本身

- **幂集**：集合所有子集组成的集合，幂集合每个元素都是集合
	- 元素个数：2^n


- **A–B**
	集合A与B的差（相对补集）
	**属于 A 但不属于 B** 的所有元素组成的集合。（用维恩图表示：A 的圆中挖去与 B 相交的部分。）

## chapter 4 关系

- **有序对** ：与顺序有关

- **关系的表示** 
	- 集合列举
	- 关系矩阵（存在的位置为1）
	- 关系图（元素作为节点，作有向边）

- **五大性质**
	- 自反性
		- 主对角线全是1
		- 每个节点有自环
	- 反自反性
		- 主对角线全是0
		- 每个节点无自环
	- 既不自反也不反自反
- 
	- 对称性
		- 对称矩阵 
		- 所有边都是双向矩阵
	- 反对称性
		- 完全无对称1
		- 不同结点间无双向边（允许单向边）
	- 既不对称也不反对称
- 
	- 传递性
		- 若有路径 a → b → c，则必须有直接边 a → c 

**运算关系**
- **复合**：R∘S={(a , c)∣∃b,(a , b)∈R∧(b , c)∈S}
- **逆关系**
- **闭包**
	依然按照几种性质来判断是什么类型的闭包
	增加有序对使得 自反/对称/传递 成立
	- 自反闭包 r
	- 对称闭包 s 
	- 传递闭包 t 

**特殊关系**
- **等价**关系
	- 自反、 对称、传递
	- 等价类
	- 集合A上的划分，确定A上一个关系
	- R就是划分所有块里元素两两排列组合的有序对的集合
- **偏序**关系
	- 自反、反对称、传递
	-  **哈斯图**：省略自环、省略由传递性可推的边、箭头朝上省略箭头
- **重点**
    - 极大元：
    - 极小元
    - 最大元/最小元
    - 上界/下界，最小上界（上确界）/最大下界（下确界）
    - **关系总结表**
    - **哈斯图判断技巧**
- **相容**关系
	- 自反
	- 对称
