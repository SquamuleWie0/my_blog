# 断舍离之后

> 2026.2.16 · 除夕夜

清空思绪，认真想了一下接下来的方向，反复权衡每个选择。

年轻，贪心。
想把能抓住的机会都攥在手里，什么都想要。
觉得好像别人有了，自己就少了，把自己搞得很累。
这一月把自己耗得差不多了。开始意识到，这段时间的思维模式是在用「年轻」换「机会」，不是个划算的买卖。

不过我觉得成长就是需要付出成本的——时间、金钱、精力、试错。
累归累，没有抱怨过，反而觉得挺值。

慢慢地，一步步地向着自己想要的"大人"模样走：工作、实习、加班、学习。

那晚加班到很晚。
我看着窗外发黄的灯，恍惚想起白天有人说，我现在看着像个"合格的大人"了。
我想，"大人"不一定就等于"合格"吧。至于标准是什么，我也说不太清。
但这段时间我确确实实体会到了 **责任** ——那种推不掉、必须担起来的责任。

我想，
成长就是，真正开始 **独当一面**。

一个人能解决所有问题，这种感觉很奇妙。
以前遇到麻烦事总想找家人帮、找朋友倾诉。现在不知不觉掌握了好多东西，开始完全把控自己的人生方向。

家里对我的发展方向向来是"散养"的。我姐一向信我，总在我需要肯定的时候让我放手去做，给我兜底。所以也导致了我做事向来敢冲。说实在的，我真不敢说每次都一定能接住，可能就是传说中的"信念感"在撑着。

我发现我反而挺享受的。
享受被高强度工作"虐"的爽感。脑子和身体都在跑，我好像反而更清醒。

握紧自己的手，一直往前走。
世界挺大的，我也不由得开始担心时间过得好快。开始期待自己会不会真的"成"。
所以每一个瞬间、每一种体验、每次相遇重逢别离，我都想好好珍惜。

2026.2.16  除夕夜
