# 年末的时候想什么

> 2025.12.28

出乎意料的这一年里，产生了我生命里重要的羁绊，结下了深厚的情感。

让我感到安心，可依赖。
同时影响着，我成为一个更加独立完整的小女孩。
教会我很多，给我带来了很多特别的感受，很多我从来没有闪过的奇思妙想。
总想好好爱护我爱的人，你总在提醒我要怎么做，好好爱护自己，那些瞬间真的很微妙，明明都没人能注意到点，就是能被你捕捉到。
就是存在这样一个人，让我可以毫无保留，无话不谈，敞开心扉，即使这样全盘托出关于我的一切也不会让我惶恐，是完全的信任存在。
我最安心的港湾，就这样刚刚好。

我从来不害怕试错、不害怕受到伤害，我甚至享受每一场意料之中或者突如其来的"冒险"。其实我从来不孤单，一直很有底气，至少想掉眼泪一定会有怀抱向我敞开，这么多年我的"后援会"一直在我身边。

我今年最大胆的一件事就是一个人偷偷直接干到哈尔滨去，小半个月辗转五座城市。

也时常一个人独自踏上旅途，在返程的航班上一个人享受内心奇妙的澎湃，感觉自己似乎开始慢慢靠近自己向往的"大人"模样。

这一年从泸州和成都出发 到 西昌 郑州 西安 贵阳 北海 南宁 重庆 济南 青岛 南昌 川西 哈尔滨 丹东 大连 威海 济南 重庆……兜兜转转最后一站回到了泸州。
心安是归途，回泸州带秦老师吃吃吃吃到合江，把我的最爱全都给qls来了一遍。还举行了与咖啡豆的世纪会晤。

从北方回来之后在程指导的利诱下（一天一包旺仔小馒头）达成了一个月的连续早睡早起flag，原来早睡早起这么美妙，貌似是近年来唯一的一段早睡早起的健康时间段。
好像也是那段时期开始，胃口变好了，心情变好了，没什么烦心事，每天都有点小期待，似乎所有让我耿耿于怀的都不纠结了。

这一年真的学会了很多很多，我明白我懂得，我其实已经很独立变得更坚毅了，我能面对解决我遇到的所有问题，但我也知道有某些人在，我也不用这么"独当一面"。

xyl又成长了。
