# 计算机网络课程笔记

##  第一章
1. 第一台电子计算机：**ENIAC**
2. 第一个计算机网络：**ARPANET**
3. **计算机网络的定义**
	- 把世界上不同地理位置独立功能的多台计算机通过通信设备和线路，实现资源共享和信息交流
4. **计算机网络发展阶段**(P23 第9题)：
	- **第1代**：面向终端的网络
	- **第2代**：以**通信子网为中心**
	- **第3代**：以**资源子网**为中心
	- **第4代**：以**Internet**为中心 

---

### 1. 计算机网络的构成

- **计算机网络硬件部分**
    - 终端设备（PC、服务器、手机等）
    - 网络设备（交换机、路由器等）
    - 传输介质（网线、光纤、无线）
- **计网软件**
    - **网络操作系统**
        - UNIX
        - Linux
        - Microsoft Windows
    - 网络协议软件（实现 TCP/IP 等通信规则）
    - 网络管理软件（监控、维护网络）
    - 网络通信软件（实现信息传递）
    - 网络应用软件（基于网络的应用系统）

---

### 2. 从功能角度划分
![[计算机网络系统结构示意图.jpg]]
计算机网络系统结构示意图
- **通信子网**
    - 由通信设备和传输介质组成
    - 承担全网的数据传输、转发、加工、转换等工作
- **资源子网**
    - 由主机及其软件资源组成
    - 负责数据处理和资源共享

---
### 3. 互联网构成
- **网络核心**    
    - 由路由器和高速链路组成
    - 负责数据转发
- **网络边缘**
    - 各种终端设备
    - 各种智能移动终端都属于
    - 负责产生和接收数据
---

结构现在是三块：
① 按组成（硬件 + 软件）  
② 按功能（通信子网 + 资源子网）  
③ 按互联网结构（核心 + 边缘）

#### 网络体系结构

**层和层之间的关系**：
下层通过接口向上层提供服务
上层依赖下层的服务与其他主机上的对等层通过通信协议来通信
实际的通信在最底层完成
**各层功能**
ISO七层模型

- **应用层**  
    提供应用程序间通信
    
- **表示层**  
    数据表示
    
- **会话层**  
    建立，维护和管理会话
    - 断点续传：断点续传 
    
- **传输层**  
    端到端的连接
    
- **网络层**  
    寻址和最佳路径选择
    
- **数据链路层**  
    编址；成帧；介质访问控制；
    
- **物理层**  
    比特流传输
