# chapter 3 向量的线性相关性
### 1. 数字型
- 向量成比例
- 行列式为0 （线性相关）
- 秩 < m （线性相关）"秩是化成“阶梯型” 非0行的数目"
### 2. 抽象型
- 抽象向量组的表示 逆向思维构造出行列式 C （C = 0 线性相关）
    - 无关组可逆阵 = 无关
    - 无关组不可逆阵 = 相关 （C = 0）
### 3. 求向量组的秩和极大无关组
- [ ] 秩：r（a1，a2，a3）=  梯形矩阵非零行的个数
- 极大无关组：阶梯拐弯处所在列向量
