# 方阵的行列式核心公式
## 1. 基本性质
- $\det(A^T) = \det(A)$（转置相等）
- $\det(kA) = k^n \det(A)$（数乘，$n$为阶数）
- 交换行/列：$\det(B) = -\det(A)$
- 行/列成比例：$\det(A) = 0$
- 行/列倍加：$\det(B) = \det(A)$

## 2. 特殊方阵
- 单位矩阵：$\det(E_n) = 1$
- 对角/上/下三角矩阵：$\det(A) = a_{11}a_{22}\cdots a_{nn}$（主对角线乘积）
- 可逆矩阵：$\det(A^{-1}) = \frac{1}{\det(A)}$（$\det(A) \neq 0$）
- 伴随矩阵：$\det(A^*) = [\det(A)]^{n-1}$

## 3. 矩阵乘积
- $\det(AB) = \det(A) \cdot \det(B)$（可推广到多个同阶方阵）



# 矩阵的秩
## 1. 核心定义
- 矩阵的秩 = 其非零子式的**最高阶数**（记为 $r(A)$）
- 推论：$r(A) \leq \min(\text{矩阵行数}, \text{矩阵列数})$

## 2. 关键性质
- 秩的不变性：$r(A) = r(A^T) = r(kA)$（$k \neq 0$，数乘非零常数）
- 秩的不等式：
  - $r(A+B) \leq r(A) + r(B)$
  - $r(AB) \leq \min(r(A), r(B))$
- 可逆矩阵性质：若 $P, Q$ 可逆，则 $r(PAQ) = r(A)$（初等变换不改变秩）

## 3. 特殊矩阵的秩
- 零矩阵：$r(O) = 0$
- 满秩矩阵（方阵）：$r(A) = n$（$n$ 为阶数）$\iff \det(A) \neq 0 \iff A$ 可逆
- 列满秩矩阵：$r(A) = \text{列数}$；行满秩矩阵：$r(A) = \text{行数}$
