### 概率统计
- 研究对象：随机现象
- 研究内容：随机现象统计规律性
- 研究方法：
### 随机事件和样本空间

**随机试验的三个特点**：
- 实验可以在相同条件下重复进行，每次试验事件A发生的概率都相同
- 试验所有结果都是 明确可知的，并却不止一个
- 每次试验只能出现一个结果

**样本空间**：随机试验E所有可能结果组成的集合，极为S或欧米嘎
研究随机现象，第一步就是建立样本空间

随机事件：样本空间的子集
基本事件：只有一个元素组成的子集，每次试验必定发生并且只可能发生一个结果
复合事件：若干个基本事件组成的随机事件
必然事件：每次试验必定会发生的事件，即样本空间
不可能事件：每次试验不可能发生的
### 随机事件的关系与运算
#### **一、事件的关系**

1. **包含关系**：A⊂B 表示事件 A 发生必然导致事件 B 发生。
2. **相等关系**：A=B 表示 A⊂B且 B⊂A，即两事件等价。
3. **互不相容（互斥）**：AB=∅表示事件 A 与事件 B 不可能同时发生。
4. **对立关系**：B=Aˉ 表示在一次试验中，A与B 有且仅有一个发生。满足：AB=∅ 且 A∪B=Ω（必然事件）。
#### **二、事件的运算**

1. **并（和）运算**：A∪B 表示 A 与 B **至少有一个**发生。
2. **交（积）运算**：A∩B （常简写为 AB） 表示 A 与 B **同时**发生。
3. **差运算**：A−B 表示 A **发生但 B**不发生。注：A−B=ABˉ。
4. **A⊂B** ：A 包含于 B （或者 B 包含 A）。 如果事件 A 发生，那么事件 B 一定发生。
5. **互不相容（互斥）**：两个事件不能同时发生，交集为空集
6. **对立事件**（互逆）：
	1. A∩B 为空集
	2. A∪B = 全集

#### **三、重要运算定律**

- **交换律**：A∪B=B∪A，A∩B=B∩A
- **结合律**：(A∪B)∪C=A∪(B∪C)，(A∩B)∩C=A∩(B∩C)
- **分配律**：A∪(B∩C)=(A∪B)∩(A∪C)，A∩(B∪C)=(A∩B)∪(A∩C)
- **德摩根定律（对偶律）**：
    - A∪B‾=Aˉ∩BˉA∪B=Aˉ∩Bˉ （和的补 = 补的交）
    - A∩B‾=Aˉ∪BˉA∩B=Aˉ∪Bˉ （交的补 = 补的和）
