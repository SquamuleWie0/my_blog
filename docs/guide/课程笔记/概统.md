> 本章不是在学习"随机变量"——而是还在**事件层面**，从已知事件概率推导未知事件概率。
>
> 核心工具就是几张概率公式：

> - 独立性 / 互斥（处理事件间关系的判定）

>
> **给一组事件和已知概率（或它们之间的独立、互斥等关系），求另一组事件的概率；或者已知结果反推原因。**

> 所以第 1 章可以理解为：**概率计算的"四则运算"基础工具箱——虽然还没引入随机变量，但所有后续章节都要反复回来用这里的公式。**

### 1.1 事件关系与运算

事件是样本空间 $\Omega$ 的子集，事件间的关系就是集合运算。

#### 基本概念

- **随机试验**：可重复、结果不唯一、事先无法确定。
- **样本空间 $\Omega$**：所有可能结果的集合。
- **随机事件**：$\Omega$ 的子集。
- **基本事件**：只有一个元素组成的子集。
- **复合事件**：若干基本事件组成的随机事件。
- **必然事件**：$\Omega$。
- **不可能事件**：$\varnothing$。

#### 事件关系

| 关系 | 记号 | 含义 |
|------|------|------|
| 包含 | $A \subset B$ | $A$ 发生则 $B$ 必发生 |
| 相等 | $A = B$ | $A \subset B$ 且 $B \subset A$ |
| 互不相容（互斥） | $A \cap B = \varnothing$ | 不能同时发生 |
| 对立事件 | $B = \overline{A}$ | $A \cap \overline{A} = \varnothing$ 且 $A \cup \overline{A} = \Omega$ |

> [!WARNING] 互斥 ≠ 对立
> 对立一定互斥，互斥不一定对立。掷骰子 $\{1\}$ 和 $\{2\}$ 互斥但不对立——还有 $\{3,4,5,6\}$。

#### 事件运算

| 运算 | 记号 | 含义 |
|------|------|------|
| 和（并） | $A \cup B$ | 至少发生一个 |
| 积（交） | $A \cap B = AB$ | 同时发生 |
| 差 | $A - B$ | $A$ 发生但 $B$ 不发生 |
| 补 | $\overline{A}$ | $A$ 不发生 |

#### 重要定律

- **交换律**：$A \cup B = B \cup A$，$A \cap B = B \cap A$
- **结合律**：$(A \cup B) \cup C = A \cup (B \cup C)$
- **分配律**：$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$
- **德摩根定律**：
  - $\overline{A \cup B} = \overline{A} \cap \overline{B}$（和的补 = 补的交）
  - $\overline{A \cap B} = \overline{A} \cup \overline{B}$（交的补 = 补的和）
  - 记忆口诀：**长杠变短杠，开口换方向**

### 1.2 概率公式（核心）

#### 1. 加法公式

$$P(A \cup B) = P(A) + P(B) - P(AB)$$

> 并集 $A \cup B$ 表示 $A$ 与 $B$ 至少一个发生；其概率为两者之和减去公共部分 $P(AB)$


当 $A,B$ 互不相容时，$P(AB)=0$，退化为 $P(A \cup B) = P(A) + P(B)$。

#### 2. 三事件容斥原理

$$P(A \cup B \cup C) = \sum P - \sum P(\text{对}) + P(ABC)$$

> 三事件并集 $A \cup B \cup C$ 的概率：奇加偶减（单个加、两两减、三个交加）


#### 3. 减法公式

$$P(A - B) = P(A) - P(AB)$$

> 差集 $A - B$ 表示 $A$ 发生而 $B$ 不发生；$P(A-B) = P(A) - P(AB)$


**特例**：$P(\overline{A}) = 1 - P(A)$。

> 对立事件 $\overline{A}$ 与 $A$ 互斥且并为 $\Omega$；$P(\overline{A}) = 1 - P(A)$


#### 4. 条件概率

$$P(A \mid B) = \frac{P(AB)}{P(B)} \quad (P(B) > 0)$$

> 条件概率 $P(A|B)$：在事件 $B$ 发生的前提下事件 $A$ 发生的概率，定义为 $P(AB)/P(B)$


**本质**：**缩小样本空间**——把参考系从 $\Omega$ 换成 $B$。

#### 5. 乘法公式

$$P(AB) = P(B) \cdot P(A \mid B) = P(A) \cdot P(B \mid A)$$

> 乘法公式：$P(AB) = P(B) \cdot P(A|B) = P(A) \cdot P(B|A)$，由条件概率变形得到


推广到 $n$ 个事件：

$$P(A_1 A_2 \cdots A_n) = P(A_1) \cdot P(A_2 \mid A_1) \cdot P(A_3 \mid A_1 A_2) \cdots$$

> n 个事件交集概率 = 逐个条件概率之积（链式法则）


#### 经典例题

**例1**：$P(A)=0.5$，$P(B)=0.4$，$P(AB)=0.2$。求 $P(A \cup B)$、$P(A-B)$、$P(B \mid A)$。

> [!example]- 答案（点击展开）
>
> **已知**：$P(A)=0.5$，$P(B)=0.4$，$P(AB)=0.2$
>
> ---
>
> **(1) 求 $P(A \cup B)$**
>
> 用**加法公式**：$P(A \cup B) = P(A) + P(B) - P(AB)$
>
> $$P(A \cup B) = 0.5 + 0.4 - 0.2 = 0.7 = \frac{7}{10}$$
>
> ---
>
> **(2) 求 $P(A-B)$**
>
> 用**减法公式**：$P(A-B) = P(A) - P(AB)$
>
> $$P(A-B) = 0.5 - 0.2 = 0.3 = \frac{3}{10}$$
>
> ---
>
> **(3) 求 $P(B \mid A)$**
>
> 用**条件概率公式**：$P(B \mid A) = \dfrac{P(AB)}{P(A)}$
>
> $$P(B \mid A) = \frac{0.2}{0.5} = \frac{2}{5}$$

**例2**：$P(A)=0.6$，$P(B)=0.5$，$P(\overline{A}\,\overline{B})=0.2$。求 $P(A \cup B)$。

> [!example]- 答案（点击展开）
>
> **已知**：$P(A)=0.6$，$P(B)=0.5$，$P(\overline{A}\,\overline{B})=0.2$
>
> **思路**：看到 $\overline{A}\,\overline{B}$（两个事件的补）→ 想 De Morgan 律
>
> **De Morgan 律**：$\overline{A} \cap \overline{B} = \overline{A \cup B}$
>
> 即 $P(\overline{A}\,\overline{B}) = P(\overline{A \cup B})$
>
> 又因为 $\overline{A \cup B}$ 与 $A \cup B$ 互补：
>
> $$P(\overline{A \cup B}) = 1 - P(A \cup B)$$
>
> 所以：
>
> $$P(A \cup B) = 1 - P(\overline{A}\,\overline{B}) = 1 - 0.2 = 0.8 = \frac{4}{5}$$

**例3**：100件中有5件次品，不放回取2件。$A$=第一件次品，$B$=第二件次品，求 $P(AB)$。

> [!example]- 答案（点击展开）
>
> **已知**：100 件中有 5 件次品，不放回取 2 件
>
> **思路**：「不放回」→ 第二次概率要变 → 用**条件概率**
>
> **Step 1**：算 $P(A)$（第一件次品的概率）
>
> $$P(A) = \frac{5}{100}$$
>
> **Step 2**：算 $P(B \mid A)$（第一件是次品后，第二件还是次品的概率）
>
> 抽走 1 件次品后剩 99 件（4 件次品 + 95 件正品）
>
> $$P(B \mid A) = \frac{4}{99}$$
>
> **Step 3**：用**乘法公式**：$P(AB) = P(A) \cdot P(B \mid A)$
>
> $$P(AB) = \frac{5}{100} \cdot \frac{4}{99} = \frac{20}{9900} = \frac{1}{495}$$

#### 易错点

| 易错点 | 正确理解 |
|--------|----------|
| 加法忘记减 $P(AB)$ | 只有互不相容时才能直接加 |
| $P(A-B) = P(A) - P(B)$ | 错误。$P(A-B) = P(A) - P(AB)$ |
| $P(A \mid B) = P(AB)$ | 忘了除以 $P(B)$ |
| $P(B)=0$ 时用条件概率 | 分母为 0，无定义 |

### 1.3 独立性

#### 独立定义

$$A \text{ 与 } B \text{ 独立} \iff P(AB) = P(A) \cdot P(B)$$

> 独立定义：$P(AB) = P(A) \cdot P(B)$ 等价于 $A$ 发生与否不影响 $B$


**等价写法**（分母 $>0$）：
- $P(A \mid B) = P(A)$
- $P(B \mid A) = P(B)$

> 独立等价条件：$A,B$ 独立 $\iff$ $P(A|B) = P(A)$，即 $B$ 发生不改变 $A$ 的概率


#### 常用结论

若 $A$ 与 $B$ 独立：
- $\overline{A}$ 与 $B$、$A$ 与 $\overline{B}$、$\overline{A}$ 与 $\overline{B}$ 也独立
- $P(A \cup B) = P(A) + P(B) - P(A)P(B)$

#### 独立 vs 互斥

|        | 互不相容               | 独立                 |
| ------ | ------------------ | ------------------ |
| 定义     | $AB = \varnothing$ | $P(AB) = P(A)P(B)$ |
| 能否同时发生 | 不能                 | 可以                 |
| 概率关系   | $P(AB) = 0$        | $P(AB) = P(A)P(B)$ |

> [!WARNING] 核心结论
> 若 $A,B$ 互斥且 $P(A) > 0, P(B) > 0$，则它们**一定不独立**。
> 因为互斥导致 $P(AB) = 0$，但 $P(A)P(B) > 0$。

#### 经典例题

**例1**：掷骰子，$A = \{1, 2\}$，$B = \{2, 4, 6\}$。判断是否独立。

> [!example]- 答案（点击展开）
>
> **Step 1**：算各项概率
>
> 样本空间有 6 个等可能结果：$\{1,2,3,4,5,6\}$
>
> - $A = \{1,2\}$，3 不在 A 里，$A \cap B = \{2\}$
> - $B = \{2,4,6\}$
>
> $$P(A) = \frac{|A|}{6} = \frac{2}{6} = \frac{1}{3}$$
> $$P(B) = \frac{|B|}{6} = \frac{3}{6} = \frac{1}{2}$$
> $$P(AB) = P(\{2\}) = \frac{1}{6}$$
>
> **Step 2**：用独立定义验证
>
> $$P(A) \cdot P(B) = \frac{1}{3} \cdot \frac{1}{2} = \frac{1}{6} = P(AB)$$
>
> 等式成立，所以 **A 与 B 独立** ✓
>
> **注意**：$A$、$B$ 有交集"2"，但**独立不等于无交集**！独立只关心概率乘积关系。

**例2**：$P(A)=0.4$，$P(B)=0.5$，$A,B$ 独立。求 $P(A \cup B)$。

> [!example]- 答案（点击展开）
>
> **已知**：$P(A)=0.4$，$P(B)=0.5$，A、B **独立**
>
> **思路**：A、B 独立 → $P(AB) = P(A) \cdot P(B)$ → 代入加法公式
>
> **加法公式**：$P(A \cup B) = P(A) + P(B) - P(AB)$
>
> 因为独立，$P(AB) = P(A) \cdot P(B) = 0.4 \times 0.5 = 0.2$
>
> 所以：
>
> $$P(A \cup B) = 0.4 + 0.5 - 0.2 = 0.7$$

### 1.4 全概率公式与贝叶斯公式

#### 1. 完备事件组（划分）

$B_1, B_2, \ldots, B_n$ 构成 $\Omega$ 的划分：
- $\bigcup_{i=1}^n B_i = \Omega$（不遗漏）
- $B_i B_j = \varnothing \; (i \neq j)$（不重叠）
- $P(B_i) > 0$

> 完备事件组：$B_1, ..., B_n$ 两两互斥且并为 $\Omega$（样本空间的一个划分）


#### 2. 全概率公式（由因推果）

$$P(A) = \sum_{i=1}^n P(B_i) \cdot P(A \mid B_i)$$

> 全概率公式（结构理解）：$P(A) = \underbrace{P(B_i)}_{\text{因的比例}} \times \underbrace{P(A|B_i)}_{\text{该因下 A 发生}}$，所有因加起来


#### 3. 贝叶斯公式（由果溯因）

$$P(B_i \mid A) = \frac{P(B_i) \cdot P(A \mid B_i)}{\sum_{j=1}^n P(B_j) \cdot P(A \mid B_j)}$$

> 贝叶斯公式（结构理解）：$P(B_k|A) = \dfrac{\overbrace{P(B_k)}^{\text{该因的比例}} \times \overbrace{P(A|B_k)}^{\text{该因下 A 发生}}}{\underbrace{\sum_i P(B_i)\,P(A|B_i)}_{\text{全概率（所有因之和）}}} = \dfrac{\text{一个加数}}{\text{总和}}$$


- $P(B_i)$：先验
- $P(B_i \mid A)$：后验
- 后验 $\propto$ 先验 × 似然

#### 经典例题

**例**：某产品由三条线 $B_1,B_2,B_3$ 生产，产量占比 50%、30%、20%，次品率 2%、3%、4%。

> [!example]- 答案（点击展开）
>
> （1）全概率——抽到次品的概率：
> $$P(A) = 0.5 \times 0.02 + 0.3 \times 0.03 + 0.2 \times 0.04 = 0.027$$
>
> （2）贝叶斯——已知是次品，来自 $B_1$ 的概率：
> $$P(B_1 \mid A) = \frac{0.5 \times 0.02}{0.027} \approx 0.37$$

### 1.5 伯努利与二项分布

> 怎么选？n 有限（抛 n 次）→ 二项 B(n,p)；n 没给（稀有计数）→ 泊松 P(λ)

#### 1. 伯努利试验

一次试验只有两种结果：$A$（成功）或 $\overline{A}$（失败）。$P(A) = p$。

#### 2. n 重伯努利试验

独立地重复 $n$ 次，每次 $p$ 不变。

**三个条件缺一不可**：独立 + 两种结果 + $p$ 不变。

**公式**：

$$P_n(k) = C_n^k \, p^k \, (1-p)^{n-k}, \quad k = 0, 1, \ldots, n$$

- $C_n^k$：选哪几次成功
- $p^k$：那些次全成功
- $(1-p)^{n-k}$：其余全失败

记作 $X \sim B(n, p)$，即二项分布。

$n=1$ 时退化为**两点分布** $B(1,p)$：

| $X$ | 0 | 1 |
|-----|---|---|
| $P$ | $1-p$ | $p$ |

#### 3. 数字特征

| 分布 | $E(X)$ | $D(X)$ |
|------|--------|--------|
| $B(1,p)$ | $p$ | $p(1-p)$ |
| $B(n,p)$ | $np$ | $np(1-p)$ |

#### 经典例题

**例1**：抛均匀硬币5次。求(1)恰好3次正面；(2)至少3次正面。

> [!example]- 答案（点击展开）
>
> **已知**：$n=5$，$p=0.5$
>
> ---
>
> **(1) 恰好 3 次正面**
>
> 用二项公式：$P_5(k) = C_n^k \cdot p^k \cdot (1-p)^{n-k}$
>
> $$P_5(3) = C_5^3 \cdot 0.5^3 \cdot 0.5^2 = 10 \times 0.125 \times 0.25 = \frac{5}{16}$$
>
> ---
>
> **(2) 至少 3 次正面**（"至少"用对立事件更简单）
>
> **思路**：$P(\geq 3) = 1 - P(\leq 2)$（$= 1 - P(0) - P(1) - P(2)$）
>
> $$P(0) = C_5^0 \cdot 0.5^5 = \frac{1}{32}$$
> $$P(1) = C_5^1 \cdot 0.5^1 \cdot 0.5^4 = \frac{5}{32}$$
> $$P(2) = C_5^2 \cdot 0.5^2 \cdot 0.5^3 = \frac{10}{32}$$
>
> $$P(\leq 2) = \frac{1 + 5 + 10}{32} = \frac{16}{32} = \frac{1}{2}$$
>
> 所以：$P(\geq 3) = 1 - \frac{1}{2} = \frac{1}{2}$

**例2**：每题4个选项，瞎蒙10题。求恰好蒙对5题的概率。

> [!example]- 答案（点击展开）
>
> **分析**：每题只有 1 个正确答案，瞎蒙猜中率 $p = 1/4 = 0.25$
>
> **已知**：$n=10$，$k=5$，$p=0.25$
>
> 用二项公式：
>
> $$P_{10}(5) = C_{10}^5 \cdot 0.25^5 \cdot 0.75^5$$
>
> 算：$C_{10}^5 = 252$，$0.25^5 = 1/1024$，$0.75^5 \approx 0.2373$
>
> $$P_{10}(5) = 252 \times \frac{1}{1024} \times 0.2373 \approx 0.058$$

**例3**：产品次品率 10%，有放回抽 3 件。求 (1) 全是正品；(2) 至少 1 件次品。

> [!example]- 答案（点击展开）
>
> **分析**：次品率 10% → 正品率 90%
>
> ---
>
> **(1) 全是正品**（即 0 件次品）
>
> 用二项公式（$n=3$，$k=0$，$p_{次品}=0.1$，但用 $0.9$ 更简单）
>
> $$P_3(0) = 0.9^3 = 0.729$$
>
> ---
>
> **(2) 至少 1 件次品**（用对立事件）
>
> **思路**：$P(\geq 1) = 1 - P(0) = 1 - P(\text{全是正品})$
>
> $$P(\geq 1) = 1 - 0.729 = 0.271$$

#### 易错点

| 易错点 | 正确理解 |
|--------|----------|
| 忘记乘 $C_n^k$ | 必须乘。成功发生在哪几次有选择 |
| 不放回也套伯努利 | 不满足独立性 |
| "至少 $k$ 次"只算一次 | 拆成 $k, k+1, \ldots, n$ 分别加 |

### 2.1 随机变量

把样本空间映射到实数的函数 $X = X(\omega)$。

|------|---------|------|
| 离散型 | 有限个或可列个 | 骰子点数、次品数 |

### 2.2 分布函数 $F(x)$

#### 定义

$$F(x) = P(X \leq x)$$

> $F(x)$ 是 $X$ 的分布函数（累积分布函数）：$F(x) = P(X \leq x)$


**本质**：**累积概率**。

#### 4 条性质（判定合法性）

- 单调不减：$x_1 < x_2 \Rightarrow F(x_1) \leq F(x_2)$
- 有界：$0 \leq F(x) \leq 1$
- 规范：$F(-\infty) = 0$，$F(+\infty) = 1$
- 右连续：$\lim_{x \to x_0^+} F(x) = F(x_0)$

#### 区间概率（4 种写法，必背）

| 区间 | 公式 | 含义 |
|---|---|---|
| $P(a < X \leq b)$ | $F(b) - F(a)$ | 减掉 a 之前的累积 |
| $P(a \leq X \leq b)$ | $F(b) - F(a-1)$ | 含 a 这一格（a-1 才能"减对"）|
| $P(a \leq X < b)$ | $F(b-1) - F(a-1)$ | 含 a 但不含 b |
| $P(a < X < b)$ | $F(b-1) - F(a)$ | 两端都不含 |

> **离散型** 必须用上面 4 种；**连续型** 这 4 种结果一样（端点概率为 0），用 $F(b) - F(a)$ 即可。

---

### 2.3 概率密度 $f(x)$（连续型专属）

#### 公式

若存在 $f(x) \geq 0$，使 $P(a < X \leq b) = \int_a^b f(x)dx$，则 $f(x)$ 为概率密度。

> $f(x)$ 是 $X$ 的概率密度；$P(a < X \leq b) = \int_a^b f(x) dx$（曲线下面积）


> $f(x)$ 本身**不是**概率，$\int f dx$ 才是。$f(x)$ 可以大于 1。

**性质**：$f(x) \geq 0$、$\int_{-\infty}^{+\infty} f = 1$、$P(X=a) = 0$。

#### 三大题型

| 题型 | 方法 |
|------|------|
| 已知 $f(x)$ 求参数 | $\int_{-\infty}^{+\infty} f(x)dx = 1$ |
| 已知 $f(x)$ 求 $F(x)$ | 分段积分，分段点处 $F$ 必须连续 |
| 已知 $F(x)$ 求 $f(x)$ | $f(x) = F'(x)$，在可导点求导 |

---

### 2.4 $F(x)$ 与 $f(x)$ 的关系

$$F(x) = \int_{-\infty}^x f(t)\,dt, \qquad f(x) = F'(x) \; (\text{在连续点})$$

|         | F(x)                | f(x)                |
| ------- | ------------------- | ------------------- |
| 中文名   | 累积分布函数           | 概率密度函数           |
| 数学本质 | 概率本身（累积）       | 概率的密度             |
| 关系     | F(x) = ∫f(t)dt（积分） | f(x) = F'(x)（求导）  |
| 互为     | f 的原函数           | F 的导函数            |
| 区间概率 | F(b) - F(a)         | ∫[a,b] f(x)dx（面积） |

**关键**：F 是 f 的原函数，f 是 F 的导函数（互为积分/求导）。

---

### 2.5 离散型：分布律

> [!NOTE] 几个概念的关系  
> **分布是总称**，表示随机变量可能取什么值，以及概率整体如何安排。
> 
> - 离散型通常用**分布律**逐点表示概率；
>     
> - 连续型通常用**概率密度 (f(x))**表示概率在各区间如何分布；
>     
> - **分布函数 (F(x))**是两者都可以使用的统一表示，表示 (P(X\le x))。
>     
> 
> 因此，分布律、概率密度和分布函数不是三种不同的分布，而是描述分布的不同方式。

#### 1. 分布律

由取值和对应概率完全确定：

| $X$ | $x_1$ | $x_2$ | $\cdots$ | $x_n$ |
|-----|-------|-------|----------|-------|
| $P$ | $p_1$ | $p_2$ | $\cdots$ | $p_n$ |

**条件**：$p_i \geq 0$，$\sum p_i = 1$。

#### 2. 求分布律步骤

1. 确定所有可能取值
2. 逐个计算 $P(X = x_i)$
3. 验证 $\sum p_i = 1$

#### 3. 已知 $F(x)$ 反求分布律

$F(x)$ 阶梯函数时，跳跃点 = 取值，跳跃高度 = 概率：

$$p_i = F(x_i) - F(x_i^-)$$

> P(X = k) = P(X ≥ k) - P(X ≥ k+1) = "含 k 的范围" − "比 k 大的范围"


#### 4. $F(x)$ 写法

$$F(x) = \sum_{x_i \leq x} p_i$$

#### 经典例题

**例1**：掷两次骰子，$X = \min(X_1, X_2)$，求分布律。

> [!example]- 答案（点击展开）
>
> 由 $P(X = k) = P(\text{都} \geq k) - P(\text{都} \geq k+1)$
>
> 其中 $P(\text{都} \geq k) = \left(\dfrac{7-k}{6}\right)^2$（两次都 $\geq k$ 的概率）
>
> 逐项计算：
> - $P(X=1) = \left(\dfrac{6}{6}\right)^2 - \left(\dfrac{5}{6}\right)^2 = \dfrac{36}{36} - \dfrac{25}{36} = \dfrac{11}{36}$
> - $P(X=2) = \left(\dfrac{5}{6}\right)^2 - \left(\dfrac{4}{6}\right)^2 = \dfrac{25}{36} - \dfrac{16}{36} = \dfrac{9}{36}$
> - $P(X=3) = \left(\dfrac{4}{6}\right)^2 - \left(\dfrac{3}{6}\right)^2 = \dfrac{16}{36} - \dfrac{9}{36} = \dfrac{7}{36}$
> - $P(X=4) = \left(\dfrac{3}{6}\right)^2 - \left(\dfrac{2}{6}\right)^2 = \dfrac{9}{36} - \dfrac{4}{36} = \dfrac{5}{36}$
> - $P(X=5) = \left(\dfrac{2}{6}\right)^2 - \left(\dfrac{1}{6}\right)^2 = \dfrac{4}{36} - \dfrac{1}{36} = \dfrac{3}{36}$
> - $P(X=6) = \left(\dfrac{1}{6}\right)^2 - 0 = \dfrac{1}{36}$
>
> 验证：$11+9+7+5+3+1 = 36$ ✓
>
> | $X$ | 1 | 2 | 3 | 4 | 5 | 6 |
> |-----|---|---|---|---|---|---|
> | $P$ | $\frac{11}{36}$ | $\frac{9}{36}$ | $\frac{7}{36}$ | $\frac{5}{36}$ | $\frac{3}{36}$ | $\frac{1}{36}$ |

**例2**：已知
$$F(x) = \begin{cases} 0 & x < 1 \\ 0.2 & 1 \leq x < 2 \\ 0.5 & 2 \leq x < 3 \\ 1 & x \geq 3 \end{cases}$$

求分布律。

> [!example]- 答案（点击展开）
>
> **思路**：$F(x)$ 是**阶梯函数**，跳一下就有一个概率
>
> 公式：$P(X = x_i) = F(x_i) - F(x_i^-)$（**跳跃的高度** = 该点概率）
>
> 其中 $F(x_i^-)$ 是 $x_i$ **左极限**（即 F 在 $x_i$ 之前那一刻的值）
>
> ---
>
> **逐项求**：
>
> - $P(X=1) = F(1) - F(1^-) = 0.2 - 0 = 0.2$  （从 0 跳到 0.2）
> - $P(X=2) = F(2) - F(2^-) = 0.5 - 0.2 = 0.3$  （从 0.2 跳到 0.5）
> - $P(X=3) = F(3) - F(3^-) = 1.0 - 0.5 = 0.5$  （从 0.5 跳到 1.0）
>
> **验证**：$0.2 + 0.3 + 0.5 = 1.0$ ✓
>
> ---
>
> **答**：分布律 $P(X=1)=0.2$，$P(X=2)=0.3$，$P(X=3)=0.5$

#### 经典例题（连续型）

**例1**：$f(x) = \begin{cases} kx & 0 \leq x \leq 2 \\ 0 & \text{其他} \end{cases}$。求 $k$、$F(x)$、$P(0.5 < X < 1.5)$。

> [!example]- 答案（点击展开）
>
> ---
>
> **(1) 求 $k$**：用概率密度规范性 $\int_{-\infty}^{\infty} f(x) dx = 1$
>
> $$1 = \int_0^2 kx \, dx = k \cdot \frac{x^2}{2} \Big|_0^2 = k \cdot 2 = 2k$$
>
> 所以 $k = \dfrac{1}{2}$
>
> ---
>
> **(2) 求 $F(x)$**：$F(x) = P(X \leq x) = \int_{-\infty}^x f(t) dt$
>
> - $x < 0$：$F(x) = 0$
> - $0 \leq x < 2$：$F(x) = \int_0^x \frac{1}{2} t \, dt = \frac{t^2}{4} \Big|_0^x = \frac{x^2}{4}$
> - $x \geq 2$：$F(x) = 1$
>
> $$F(x) = \begin{cases} 0 & x < 0 \\ x^2/4 & 0 \leq x < 2 \\ 1 & x \geq 2 \end{cases}$$
>
> ---
>
> **(3) 求 $P(0.5 < X < 1.5)$**：用 $F$ 差
>
> $$P(0.5 < X < 1.5) = F(1.5) - F(0.5) = \frac{1.5^2}{4} - \frac{0.5^2}{4} = \frac{2.25 - 0.25}{4} = \frac{2}{4} = 0.5$$

**例2**：$f(x) = \begin{cases} 2x & 0 < x < 1 \\ 0 & \text{其他} \end{cases}$。求 $E(X)$ 和 $D(X)$。

> [!example]- 答案（点击展开）
>
> **公式**：$E(X) = \int x f(x) dx$，$D(X) = E(X^2) - [E(X)]^2$，$E(X^2) = \int x^2 f(x) dx$
>
> ---
>
> **(1) 求 $E(X)$**
>
> $$E(X) = \int_0^1 x \cdot 2x \, dx = \int_0^1 2x^2 \, dx = \left.\frac{2x^3}{3}\right|_0^1 = \frac{2}{3}$$
>
> ---
>
> **(2) 求 $E(X^2)$**
>
> $$E(X^2) = \int_0^1 x^2 \cdot 2x \, dx = \int_0^1 2x^3 \, dx = \left.\frac{2x^4}{4}\right|_0^1 = \frac{1}{2}$$
>
> ---
>
> **(3) 求 $D(X)$**：$D(X) = E(X^2) - [E(X)]^2$
>
> $$D(X) = \frac{1}{2} - \left(\frac{2}{3}\right)^2 = \frac{1}{2} - \frac{4}{9} = \frac{9 - 8}{18} = \frac{1}{18}$$

#### 易错点

| 易错点 | 正确理解 |
|--------|----------|
| 漏取值 | 穷举后用 $\sum p_i = 1$ 验证 |
| $F(x_i^-)$ 误当 $F(x_{i-1})$ | 严格算左极限 |
| 离散型 $F(x)$ 不右连续 | 阶梯函数也是右连续的 |
| $f(x)$ 是概率 | $f(x)dx$ 才是概率微元 |
| $F(x)$ 在分段点中断 | $F(x)$ 处处连续 |
| 积分忘分段 | $f(x)$ 分段定义，积分也必须分段 |

## 三、常见分布


> 考纲对应：考点 5（六大分布的 E/D、密度、分布函数）
> 参考页码：P99 六大分布



> [!NOTE] 第 3 章到底在学什么
> 第 3 章不是在学习一套新的计算方法，而是把常见随机现象整理成六个现成模型。
>
> 考试题本质上就是：
>
> **根据题意识别属于哪种分布，找出参数，再直接使用对应的概率公式、期望和方差。**
>
> - 计次数：两点、二项、泊松分布；
> - 看连续取值或等待时间：均匀、指数分布；
> - 数据集中在平均值附近：正态分布。
>
> 所以第 3 章可以理解为：**识别现实场景属于哪个固定模板，然后套用模板。**


### 3.1 离散分布

#### 1. 两点（0-1）分布 $B(1,p)$

- 分布律：$P(X=0)=1-p,\; P(X=1)=p$
- $E(X) = p$，$D(X) = p(1-p)$

#### 2. 二项分布 $B(n,p)$

- 分布律：$P(X=k) = C_n^k p^k (1-p)^{n-k}$
- $E(X) = np$，$D(X) = np(1-p)$

#### 3. 泊松分布 $P(\lambda)$

- 分布律：$P(X=k) = \dfrac{\lambda^k e^{-\lambda}}{k!}$
- $E(X) = \lambda$，$D(X) = \lambda$
- 含义：**稀有事件**计数。

---

### 3.2 连续分布

#### 1. 均匀分布 $U(a,b)$

**密度**：
$$f(x) = \begin{cases} \dfrac{1}{b-a} & a \leq x \leq b \\ 0 & \text{其他} \end{cases}$$

**分布函数**：
$$F(x) = \begin{cases} 0 & x < a \\ \dfrac{x-a}{b-a} & a \leq x \leq b \\ 1 & x > b \end{cases}$$

**区间概率**：$[c,d] \subseteq [a,b]$ 时 $P(c < X < d) = \dfrac{d-c}{b-a}$。

概率只和区间长度成正比，与位置无关——这就是"均匀"的含义。

**数字特征**：$E(X) = \dfrac{a+b}{2}$，$D(X) = \dfrac{(b-a)^2}{12}$

#### 2. 指数分布 $E(\lambda)$

- 密度：$f(x) = \lambda e^{-\lambda x}$，$x > 0$
- $E(X) = \dfrac{1}{\lambda}$，$D(X) = \dfrac{1}{\lambda^2}$
- 无记忆性：$P(X>s+t \mid X>s) = P(X>t)$

> λ 在泊松算"次数"、在指数算"频率"，**互为倒数**（泊松 λ=5 对应指数 λ=1/5）。无记忆性 = "忘了过去"，只关心未来。

#### 3. 正态分布 $N(\mu,\sigma^2)$（重点）

> [!WARNING] 必看：$N(\mu, \sigma^2)$ 的两个参数是什么（**别和 $B(n, p)$ 搞混**）
>
> **$N$ 的两个参数都是"分布的统计量"**——描述分布长什么样：
>
> | 位置 | 符号 | 含义 | 直接读出什么 |
> |------|------|------|------------|
> | **第一个** | $\mu$ | **均值**（中心位置）| $E(X) = \mu$ |
> | **第二个** | $\sigma^2$ | **方差**（**注意：不是标准差！**）| $D(X) = \sigma^2$ |
>
> **直接读 $E(X)$ 和 $D(X)$**：
> - 看到 $N(0, 1)$：$E(X) = 0$，$D(X) = 1$（标准正态）
> - 看到 $N(3, 4)$：$E(X) = 3$，$D(X) = 4$（**$\sigma = \sqrt{4} = 2$，标准差要开根号**）
>
> **对比 $B(n, p)$——绝对别混**：
>
> | 分布 | 第一个参数 | 第二个参数 | 描述的是 |
> |------|----------|----------|---------|
> | $N(\mu, \sigma^2)$ | $\mu$ = 均值 | $\sigma^2$ = 方差 | **分布本身**（中心、胖瘦）|
> | $B(n, p)$ | $n$ = 试验次数 | $p$ = 成功概率 | **试验条件**（做几次、成功率）|
>
> **记忆口诀**：
> - $N$ 是"长什么样"（**统计量**：均值、方差）
> - $B$ 是"怎么试验"（**试验参数**：次数、概率）

**密度**：
$$f(x) = \frac{1}{\sqrt{2\pi}\,\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

- $\mu$：均值，决定中心位置
- $\sigma^2$：方差，决定胖瘦
- 图像为钟形，关于 $x=\mu$ 对称

**标准化**：
$$Z = \frac{X - \mu}{\sigma} \sim N(0, 1)$$

标准正态密度 $\varphi(x) = \frac{1}{\sqrt{2\pi}} e^{-x^2/2}$，分布函数记为 $\Phi(x)$。

**$\Phi(x)$ 关键性质**：
- $\Phi(0) = 0.5$
- $\Phi(-x) = 1 - \Phi(x)$（对称性）

**概率计算**：
$$P(a < X \leq b) = \Phi\!\left(\frac{b-\mu}{\sigma}\right) - \Phi\!\left(\frac{a-\mu}{\sigma}\right)$$

三步：标准化 → 查表 → 相减。

**3σ 原则**：

| 区间 | 概率 |
|------|------|
| $\mu \pm \sigma$ | $\approx 68.3 \%$ |
| $\mu \pm 2\sigma$ | $\approx 95.5 \%$ |
| $\mu \pm 3\sigma$ | $\approx 99.7 \%$ |

**数字特征**：$E(X) = \mu$，$D(X) = \sigma^2$

正态的线性变换仍为正态：$aX+b \sim N(a\mu+b,\, a^2\sigma^2)$。

**概率计算 4 大公式**（标准化 → 查表 → 计算）：

| 类型 | 公式 | 备注 |
|---|---|---|
| **区间** | $P(a < X \leq b) = \underbrace{\Phi\!\left(\dfrac{b-\mu}{\sigma}\right)}_{\text{右端累积}} - \underbrace{\Phi\!\left(\dfrac{a-\mu}{\sigma}\right)}_{\text{左端累积}}$ | 两面积相减 |
| **单边左** | $P(X < a) = \underbrace{\Phi\!\left(\dfrac{a-\mu}{\sigma}\right)}_{\text{左侧累积面积}}$ | 小于直接 $\Phi$ |
| **单边右** | $P(X > a) = 1 - \underbrace{\Phi\!\left(\dfrac{a-\mu}{\sigma}\right)}_{\text{左侧面积}}$ | 大于 $=1-\Phi$（互补）|
| **反求参数** | 给概率反查 $\Phi^{-1}$ → 解方程 | 见例 3 |

> 小于直接 $\Phi$，大于 $1-\Phi$（互补关系）。

#### 经典例题

**例1**：$X \sim N(3, 4)$，求 $P(1 < X < 5)$。

> [!example]- 答案（点击展开）
>
> **解**：
>
> (1) **已知**：$X \sim N(3, 4)$，故 $\mu = 3$，$\sigma^2 = 4$，$\sigma = 2$
>
> (2) **标准化**（$Z = \dfrac{X - \mu}{\sigma}$）：
> $$Z_1 = \frac{1 - 3}{2} = -1, \quad Z_2 = \frac{5 - 3}{2} = 1$$
>
> (3) **套公式**（$P(a < X < b) = \Phi(Z_2) - \Phi(Z_1)$）：
> $$P(1 < X < 5) = \Phi(1) - \Phi(-1)$$
>
> (4) **用对称性**（$\Phi(-x) = 1 - \Phi(x)$）：
> $$\Phi(-1) = 1 - \Phi(1) = 1 - 0.8413 = 0.1587$$
>
> (5) **代入计算**：
> $$P(1 < X < 5) = 0.8413 - 0.1587 = 0.6826$$
>
> (6) **答**：$P(1 < X < 5) \approx 0.6826$

**例2**：$X \sim N(10, 9)$，求 $P(X > 16)$ 和 $P(4 < X < 13)$。

> [!example]- 答案（点击展开）
>
> **已知**：$X \sim N(10, 9)$，故 $\mu = 10$，$\sigma^2 = 9$，$\sigma = 3$
>
> ---
>
> **求 (1)** $P(X > 16)$
>
> (1) **标准化**：$Z = \dfrac{X - \mu}{\sigma} = \dfrac{16 - 10}{3} = 2$
>
> (2) **套公式**（$P(X > a) = 1 - \Phi(Z)$）：
> $$P(X > 16) = 1 - \Phi(2)$$
>
> (3) **查表**：$\Phi(2) = 0.9772$
>
> (4) **代入**：
> $$P(X > 16) = 1 - 0.9772 = 0.0228$$
>
> (5) **答**：$P(X > 16) \approx 0.0228$
>
> ---
>
> **求 (2)** $P(4 < X < 13)$
>
> (1) **标准化**：
> $$Z_1 = \frac{4 - 10}{3} = -2, \quad Z_2 = \frac{13 - 10}{3} = 1$$
>
> (2) **套公式**（$P(a < X < b) = \Phi(Z_2) - \Phi(Z_1)$）：
> $$P(4 < X < 13) = \Phi(1) - \Phi(-2)$$
>
> (3) **查表 + 对称性**：
> $$\Phi(1) = 0.8413, \quad \Phi(-2) = 1 - \Phi(2) = 1 - 0.9772 = 0.0228$$
>
> (4) **代入**：
> $$P(4 < X < 13) = 0.8413 - 0.0228 = 0.8185$$
>
> (5) **答**：$P(4 < X < 13) \approx 0.8185$

**例3**：考试成绩 $X \sim N(72, \sigma^2)$，已知10%学生超过84分，求 $\sigma$。

> [!example]- 答案（点击展开）
>
> **已知**：$X \sim N(72, \sigma^2)$，$\mu = 72$，$P(X > 84) = 0.1$
>
> (1) **标准化**：$Z = \dfrac{84 - 72}{\sigma} = \dfrac{12}{\sigma}$
>
> (2) **套公式**（$P(X > a) = 1 - \Phi(Z)$）：
> $$P(X > 84) = 1 - \Phi\!\left(\frac{12}{\sigma}\right) = 0.1$$
>
> (3) **反查**（移项）：
> $$\Phi\!\left(\frac{12}{\sigma}\right) = 1 - 0.1 = 0.9$$
> $$\frac{12}{\sigma} = \Phi^{-1}(0.9)$$
>
> (4) **查表**：$\Phi^{-1}(0.9) \approx 1.28$
>
> (5) **代入解方程**：
> $$\frac{12}{\sigma} = 1.28$$
> $$\sigma = \frac{12}{1.28} \approx 9.375$$
>
> (6) **答**：$\sigma \approx 9.375$

#### 易错点

| 易错点 | 正确理解 |
|--------|----------|
| 标准化除以 $\sigma^2$ | 除以**标准差** $\sigma$ |
| $\Phi(-x) = -\Phi(x)$ | $\Phi(-x) = 1 - \Phi(x)$ |
| $\Phi(x)$ 是上侧概率 | $\Phi(x)$ 是下侧 $P(X \leq x)$ |
| 非标准正态直接查表 | 必须先标准化 |

---

### 3.3 变量函数的分布

> 已知 $X$ 的分布，求 $Y = g(X)$ 的分布

#### 0. 基础知识（必学）

**单调性判断**（决定能否用公式法）：
- 单调递增 ↗：$y = 2x+1$、$y = x^3$
- 单调递减 ↘：$y = -x$、$y = 1/x$
- 不单调 ∽：$y = x^2$、$y = \sin x$（不能用公式法）

**反函数 h(y) 求法**（以 $y = 2x+1$ 为例）：
1. 把 $y$ 当已知，$x$ 当未知
2. 解出 $x = (y-1)/2$
3. 写出 $h(y) = (y-1)/2$

**$h'(y)$ 求法**：对 $h(y)$ 求导（把 $y$ 当自变量）

**$|h'(y)|$ 取绝对值原因**：保证 $f_Y(y) \geq 0$（概率密度非负）

#### 1. 分布函数法（通用）

1. $F_Y(y) = P(Y \leq y) = P(g(X) \leq y)$
2. 解出 $X$ 的范围，用 $f_X$ 积分
3. **按 $y$ 的不同区间分段讨论**
4. $f_Y(y) = F_Y'(y)$

#### 2. 公式法（重点掌握）

$$f_Y(y) = f_X(h(y)) \cdot |h'(y)|$$

其中 $Y = g(X)$，$x = h(y)$ 是 $g$ 的反函数。

**步骤**：
1. 求 $g(x)$ 的值域 = $Y$ 的取值范围
2. 在 $X$ 的值域上，验证 $g(x)$ **单调**
3. 求反函数 $x = h(y)$ 和 $|h'(y)|$
4. 代入公式得 $f_Y(y)$，值域外补 0

#### 经典例题

**例1**：$X \sim U(0, 1)$，$Y = 2X + 1$。求 $f_Y(y)$。

> [!example]- 答案（点击展开）
>
> **解**：
>
> (1) **判断单调性**：$g(x) = 2x+1$ 是线性函数，**单调递增** ✓（可用公式法）
>
> (2) **求 g 的值域**：$x \in [0,1]$，所以 $y = 2x+1 \in [1, 3]$
>
> (3) **求反函数和导数**：
> $$y = 2x + 1 \Rightarrow y - 1 = 2x \Rightarrow h(y) = \frac{y-1}{2}, \quad h'(y) = \frac{1}{2}$$
>
> (4) **代入公式**：
> $$f_Y(y) = f_X\!\left(\frac{y-1}{2}\right) \cdot \left|\frac{1}{2}\right| = f_X\!\left(\frac{y-1}{2}\right) \cdot \frac{1}{2}$$
>
> 因为 $X \sim U(0,1)$，$f_X(x) = 1$（在 [0,1] 内）
> $$f_Y(y) = 1 \cdot \frac{1}{2} = \frac{1}{2}, \quad y \in (1, 3)$$
>
> (5) **答**：$Y \sim U(1, 3)$

**例2**：$X \sim N(0, 1)$，$Y = X^2$。求 $f_Y(y)$。

> [!example]- 答案（点击展开）
>
> **解**：
>
> (1) **判断单调性**：$g(x) = x^2$ **不是单调**（先减后增），**不能用公式法**！
> → 用**分布函数法**
>
> (2) **写 F_Y(y)**：
> $$F_Y(y) = P(X^2 \leq y) = P(-\sqrt{y} \leq X \leq \sqrt{y}) = \Phi(\sqrt{y}) - \Phi(-\sqrt{y})$$
>
> (3) **化简**（用 $\Phi(-x) = 1 - \Phi(x)$）：
> $$F_Y(y) = \Phi(\sqrt{y}) - [1 - \Phi(\sqrt{y})] = 2\Phi(\sqrt{y}) - 1, \quad y > 0$$
>
> (4) **求导得 $f_Y(y)$**：
> $$f_Y(y) = 2\phi(\sqrt{y}) \cdot \frac{1}{2\sqrt{y}} = \frac{1}{\sqrt{2\pi y}} e^{-y/2}, \quad y > 0$$
>
> (5) **答**：$f_Y(y) = \dfrac{1}{\sqrt{2\pi y}} e^{-y/2}$，$y > 0$（这就是 $\chi^2$ 分布！）

**例3**：$X \sim U(0, 2)$，$Y = \begin{cases} 1 & X < 1 \\ 2 & X \geq 1 \end{cases}$。求 $Y$ 的分布律。

> [!example]- 答案（点击展开）
>
> **解**：
>
> (1) **分析 Y**：$Y$ 只取 1 或 2，是**离散型**
>
> (2) **算各概率**：
> $$P(Y=1) = P(X < 1) = \int_0^1 \frac{1}{2} dx = \frac{1}{2}$$
> $$P(Y=2) = P(X \geq 1) = \int_1^2 \frac{1}{2} dx = \frac{1}{2}$$
>
> (3) **答**：$P(Y=1) = P(Y=2) = \dfrac{1}{2}$

#### 易错点

| 易错点 | 正确理解 |
|--------|----------|
| 不按 $y$ 分段 | 不同 $y$ 对应不同的 $X$ 范围 |
| 忘绝对值 | $f_Y = f_X(h(y)) \cdot \lvert h'(y) \rvert$ |
| $g$ 不单调时仍用公式法 | 必须改用分布函数法 |
| 忘求 g 的值域 | $f_Y$ 必须带区间 |

## 四、多维随机变量（仅离散型）

> 考纲对应：**考点 1**（二维离散型 → 边缘分布/期望/协方差/独立性）
> 参考页码：P61 基础 1 题、P62 提高 1 题、P68 基础 5 题、P69 提高 1 题、P85 总复习 (3) 题

> [!NOTE] 第 4 章到底在学什么
> 第 2 章研究的是**一个**随机变量，第 4 章要研究**两个**随机变量的联合行为。
>
> 核心载体是**联合分布律**——一张二维表格，每一格是 $(X=x_i, Y=y_j)$ 同时发生的概率。
>
> 考试题本质上就是：
>
> **给一张联合分布律表 → 求边缘分布（行/列和）、条件分布（固定一列后归一化）、$E(X)$、$E(Y)$、$\text{Cov}$、判断独立性。**
>
> 所以第 4 章可以理解为：**用一张二维表格把两个变量的关系完整描述出来，然后再从这张表里"挖"出各种信息——边缘、条件、期望、协方差、独立性全都来自这张表。**

### 4.1 联合分布律

$$P(X = x_i, Y = y_j) = p_{ij}$$

| $X \setminus Y$ | $y_1$ | $y_2$ | $y_3$ |
|-----------------|-------|-------|-------|
| $x_1$ | $p_{11}$ | $p_{12}$ | $p_{13}$ |
| $x_2$ | $p_{21}$ | $p_{22}$ | $p_{23}$ |

**规范性**：$\sum_i \sum_j p_{ij} = 1$。

---

### 4.2 边缘分布律

**求边缘分布的 2 步走**：

| 步骤 | 操作 | 含义 |
|------|------|------|
| ① | 算行和 $p_{i\cdot} = \sum_j p_{ij}$ | $P(X = x_i)$ = 第 $i$ 行加起来 |
| ② | 算列和 $p_{\cdot j} = \sum_i p_{ij}$ | $P(Y = y_j)$ = 第 $j$ 列加起来 |

**操作**：表格**右边加一列行和**，**下边加一行列和**。

**验证**：行和加起来 = 1，列和加起来 = 1。

**口诀**：**行和给 X，列和给 Y**——把谁的下标消掉（用点替代），那个变量就被"边缘化"掉了。

---

### 4.3 条件分布律

**求条件分布 $P(X \mid Y=y_j)$ 的 3 步走**：

| 步骤 | 操作 | 含义 |
|------|------|------|
| ① | 选定一列（固定 $Y = y_j$）| 锁定要条件的变量 |
| ② | 算列和 $p_{\cdot j}$（即 $P(Y=y_j)$）| 拿到分母 |
| ③ | 把这一列每个 $p_{ij}$ **除以 $p_{\cdot j}$** | 得到 $P(X=x_i \mid Y=y_j)$ |

**公式**：$P(X = x_i \mid Y = y_j) = \dfrac{p_{ij}}{p_{\cdot j}}$（即"列内归一化"）

**口诀**：**固定一列，除以列和**——一列概率本来加总到 $P(Y=y_j)$，除一下就归一化成合法的条件分布。

**验证**：条件分布的各概率之和 = 1（公式自动保证，不用单独算）。

> 原始列是"绝对概率"（加总 = $P(Y=y_j)$），除以列和后变成"相对概率"（加总 = 1）。

---

### 4.4 独立性判定（离散型）

**判别独立性的 4 步走**：

| 步骤 | 操作 | 含义 |
|------|------|------|
| ① | 算行和 $p_{i\cdot}$（每行加起来）| 得到 $X$ 的边缘 |
| ② | 算列和 $p_{\cdot j}$（每列加起来）| 得到 $Y$ 的边缘 |
| ③ | 算 $p_{i\cdot} \cdot p_{\cdot j}$ | 期望的联合概率 |
| ④ | 逐格比对 $p_{ij}$ vs 乘积 | 全等 → **独立**；有一个不等 → **不独立** |

**公式**（第 ④ 步的判别标准）：$p_{ij} = p_{i\cdot} \cdot p_{\cdot j}$（对所有 $i, j$ 都成立）

**关键口诀**：**一个反例就够**——只要找到一格不等，就判不独立，不用全验完。

**简例（不独立）**：

| $X \setminus Y$ | 0 | 1 | 行和 |
|---|---|---|---|
| 0 | 0.3 | 0.2 | 0.5 |
| 1 | 0.1 | 0.4 | 0.5 |
| **列和** | **0.4** | **0.6** | 1 |

只验 (0, 0) 一格：$p_{00} = 0.3$ vs $0.5 \times 0.4 = 0.2$——不等 → **不独立**（其余格子不用算了）。

---

### 4.5 重要推论

- $X,Y$ 独立 $\Rightarrow E(XY) = E(X)E(Y)$
- $X,Y$ 独立 $\Rightarrow \text{Cov}(X,Y) = 0$
- $X,Y$ 独立 $\Rightarrow D(X \pm Y) = D(X) + D(Y)$

> [!WARNING] 不相关 ≠ 独立
> $\text{Cov} = 0$ 不能反推独立。**唯一例外**：二维正态分布中 $\rho = 0 \iff$ 独立。

---

### 4.6 考点 1 解题套路（4 步走）

1. **写联合分布表**：列出 $(X,Y)$ 所有取值对应的概率 $p_{ij}$
2. **求边缘分布**：行/列求和
   - $P(X=x_i) = \sum_j p_{ij}$，记为 $p_{i\cdot}$
   - $P(Y=y_j) = \sum_i p_{ij}$，记为 $p_{\cdot j}$
3. **算期望**：
   - $E(X) = \sum_i \sum_j x_i \cdot p_{ij}$
   - $E(XY) = \sum_i \sum_j x_i y_j \cdot p_{ij}$
4. **协方差 + 独立性判断**：
   - $\text{Cov}(X,Y) = E(XY) - E(X)E(Y)$
   - **独立 ⇔ 对所有 $i,j$：$p_{ij} = p_{i\cdot} \cdot p_{\cdot j}$**

---

### 经典例题

**例1**：联合分布律如下，求边缘分布、条件分布、$E(X)$、$E(Y)$、$\text{Cov}(X,Y)$，判断独立性。

| $X \setminus Y$ | 0 | 1 |
|---|---|---|
| 0 | 0.3 | 0.2 |
| 1 | 0.1 | 0.4 |

> [!example]- 答案（点击展开）
>
> **边缘分布**：$P(X=0)=0.5$，$P(X=1)=0.5$；$P(Y=0)=0.4$，$P(Y=1)=0.6$
>
> **条件分布**：
> $P(X=1 \mid Y=0) = \frac{0.1}{0.4} = 0.25$
> $P(X=1 \mid Y=1) = \frac{0.4}{0.6} = \frac{2}{3}$
>
> **期望**：
> $E(X) = 0 \times 0.5 + 1 \times 0.5 = 0.5$
> $E(Y) = 0 \times 0.4 + 1 \times 0.6 = 0.6$
> $E(XY) = 1 \times 1 \times 0.4 = 0.4$
>
> **协方差**：
> $\text{Cov}(X,Y) = E(XY) - E(X)E(Y) = 0.4 - 0.5 \times 0.6 = 0.1$
>
> **独立性**：验证 $p_{00} = 0.3 \ne p_{0\cdot} \cdot p_{\cdot 0} = 0.5 \times 0.4 = 0.2$，**不独立**。

**例2**：判断独立性。

| $X \setminus Y$ | 0 | 1 |
|---|---|---|
| 0 | 0.15 | 0.35 |
| 1 | 0.15 | 0.35 |

> [!example]- 答案（点击展开）
>
> **Step 1**：求边缘分布
>
> - $P(X=0) = 0.15 + 0.35 = 0.5$，$P(X=1) = 0.15 + 0.35 = 0.5$
> - $P(Y=0) = 0.15 + 0.15 = 0.3$，$P(Y=1) = 0.35 + 0.35 = 0.7$
>
> **Step 2**：逐项验证 $p_{ij} = p_{i\cdot} \cdot p_{\cdot j}$
>
> - $p_{00} = 0.15$，$p_{0\cdot} \cdot p_{\cdot 0} = 0.5 \times 0.3 = 0.15$ ✓
> - $p_{01} = 0.35$，$p_{0\cdot} \cdot p_{\cdot 1} = 0.5 \times 0.7 = 0.35$ ✓
> - $p_{10} = 0.15$，$p_{1\cdot} \cdot p_{\cdot 0} = 0.5 \times 0.3 = 0.15$ ✓
> - $p_{11} = 0.35$，$p_{1\cdot} \cdot p_{\cdot 1} = 0.5 \times 0.7 = 0.35$ ✓
>
> **四个格子都满足 → X、Y 独立** ✓

#### 易错点

| 易错点 | 正确理解 |
|--------|----------|
| 边缘行列搞反 | $X$ 对 $j$ 求和（行和），$Y$ 对 $i$ 求和（列和） |
| 独立 ≠ 协方差=0 | 独立 ⇒ 不相关，反之不成立（仅二维正态等价） |
| 离散型判独立 | 必须**逐项验证** $p_{ij} = p_{i\cdot} \cdot p_{\cdot j}$ |
| 漏组合 | 穷举后用 $\sum\sum p_{ij}=1$ 验证规范性 |

## 五、数字特征

> 考纲对应：**考点 6**（期望方差性质、协方差相关系数性质）+ 客观题 4
> 参考页码：P93（期望/方差性质）、P99（六大分布 E/D）、P102 例 4.18、P103（协方差/相关系数性质）、P106-8 题

> [!NOTE] 第 4 章到底在学什么
> 第 2、3 章都在回答"随机变量的概率怎么分布"，第 5 章要回答的是另一个问题：**用一个或几个数字概括整个分布**。
>
> 三个核心数字标签：
>
> - $E(X)$：描述**中心位置**（均值在哪）
> - $D(X)$：描述**离散程度**（胖瘦）
> - $\text{Cov}(X,Y)$、$\rho$：描述**两变量同向变化的程度**
>
> 考试题本质上就是：
>
> **已知分布（分布律或密度），求 $E(X)$、$D(X)$；已知联合分布，求 $\text{Cov}$、$\rho$、判断独立性；或者用性质（线性、独立性）做化简计算。**
>
> 所以第 5 章可以理解为：**给随机变量"贴数字标签"——一个标签 $E(X)$ 描述中心、一个标签 $D(X)$ 描述胖瘦、一个标签 $\rho$ 描述两变量是否同向变化。**

### 5.1 期望 $E(X)$ 与方差 $D(X)$

#### 1. 数学期望 $E(X)$（加权平均）

| 类型  | 公式                                           |
| --- | -------------------------------------------- |
| 离散  | $E(X) = \sum x_i p_i$                        |
| 连续  | $E(X) = \int_{-\infty}^{+\infty} x f(x)\,dx$ |

>  本质：**加权平均**——以概率为权重的加权平均，反映"中心位置"。

> [!NOTE] 如何理解“概率作为权重”
> 某个取值出现的概率越大，它对期望的影响就越大，期望会更靠近这个取值。
>
> 但概率变大不一定会让期望的数值变大：
> - 较大取值的概率增加，期望通常增大；
> - 较小或负数取值的概率增加，期望反而可能减小。


**性质**：
- $E(c) = c$
- $E(cX) = cE(X)$
- $E(X \pm Y) = E(X) \pm E(Y)$（不需要独立）
- $E(XY) = E(X)E(Y)$（需要独立）


#### 2. 方差 $D(X)$

$$D(X) = E[(X - E(X))^2]$$

>  本质：方差 = **平方均值 − 均值的平方**

实际计算必用：

$$D(X) = E(X^2) - [E(X)]^2$$

> 方差计算式：$D(X) = E(X^2) - [E(X)]^2$（更易算的等价形式）

标准差 $\sigma(X) = \sqrt{D(X)}$。

$E(X^2) = \sum x_i^2 p_i$（离散）或 $\int x^2 f(x)dx$（连续）。

**性质**：
- $D(c) = 0$
- $D(cX) = c^2 D(X)$
- $X,Y$ 独立：$D(X \pm Y) = D(X) + D(Y)$
- 不独立：$D(X \pm Y) = D(X) + D(Y) \pm 2\text{Cov}(X,Y)$

> [!WARNING] $D(2X) = 4D(X)$，不是 $2D(X)$。系数必须平方。

#### 3. 离散 vs 连续 E(X)/D(X) 公式对比

| 量 | 离散型 | 连续型 |
|---|---|---|
| E(X) | $\sum x_i p_i$ | $\int x f(x) dx$ |
| E(X²) | $\sum x_i^2 p_i$ | $\int x^2 f(x) dx$ |
| D(X) | $E(X^2) - [E(X)]^2$ | 同上（公式一样）|

> 本质：E(X) = 加权平均；D(X) = 平方均值 − 均值平方

> [!NOTE] 离散型与连续型的区别
> - 离散型随机变量按一个个具体取值计算，每个点可以具有非零概率，因此使用：
>   $$
>   E(X)=\sum x_i p_i
>   $$
> - 连续型随机变量中，任意单个点的概率都严格等于 $0$：
>   $$
>   P(X=a)=0
>   $$
>   概率需要在区间上计算，因此使用概率密度积分：
>   $$
>   E(X)=\int_{-\infty}^{+\infty}xf(x)\,dx
>   $$
>
> 注意：$f(a)$ 是概率密度，不是 $P(X=a)$；区间上的积分才是概率。

---

### 5.2 常见分布的 E/D 速查

> [!WARNING] ⚠️ 这张表**考试不发公式表，必须背下来**！
> 6 个分布 × 2 个数字（$E$、$D$）= 12 个值，这是矩估计、协方差计算、5.3 协方差等所有题的基础。
> **默写标准**：看到 $N(0,1)$ 立刻写 $E=0, D=1$；看到 $U(0,\theta)$ 立刻写 $E=\theta/2, D=\theta^2/12$——**不需要想**。

| 分布 | $E(X)$ | $D(X)$ | 如何识别 |
|------|--------|--------|---------|
| 两点分布 $B(1,p)$ | $p$ | $p(1-p)$ | 一次 0/1 试验（$n=1$ 的二项特例）|
| 二项分布 $B(n,p)$ | $np$ | $np(1-p)$ | n 次独立 0/1 重复（**题给了 $n$**）|
| 泊松分布 $P(\lambda)$ | $\lambda$ | $\lambda$ | 稀有事件计数（**没给 $n$，只给均值**）|
| 均匀分布 $U(a,b)$ | $\frac{a+b}{2}$ | $\frac{(b-a)^2}{12}$ | 在 $[a,b]$ 内**等可能**取值 |
| 指数分布 $E(\lambda)$ | $\frac{1}{\lambda}$ | $\frac{1}{\lambda^2}$ | 等待**时间**（下个事件多久到）|
| 正态分布 $N(\mu,\sigma^2)$ | $\mu$ | $\sigma^2$ | 题目标 $X \sim N(\mu,\sigma^2)$ |

#### 经典例题

**例1**：$X$ 分布律 $P(X=-1)=0.2$，$P(X=0)=0.5$，$P(X=2)=0.3$。求 $E(X)$ 和 $D(X)$。

> [!example]- 答案（点击展开）
>
> **公式**：$E(X) = \sum x_i p_i$，$D(X) = E(X^2) - [E(X)]^2$
>
> ---
>
> **(1) 求 $E(X)$**（每个 $x$ 乘对应概率再加总）
>
> $$E(X) = (-1) \times 0.2 + 0 \times 0.5 + 2 \times 0.3 = -0.2 + 0 + 0.6 = 0.4$$
>
> ---
>
> **(2) 求 $E(X^2)$**（把 $X$ 换成 $X^2$）
>
> $$E(X^2) = (-1)^2 \times 0.2 + 0^2 \times 0.5 + 2^2 \times 0.3 = 0.2 + 0 + 1.2 = 1.4$$
>
> ---
>
> **(3) 求 $D(X)$**：$D(X) = E(X^2) - [E(X)]^2$
>
> $$D(X) = 1.4 - (0.4)^2 = 1.4 - 0.16 = 1.24$$

**例2**：$f(x) = \begin{cases} 3x^2 & 0 < x < 1 \\ 0 & \text{其他} \end{cases}$。求 $E(X)$ 和 $D(X)$。

> [!example]- 答案（点击展开）
>
> **公式**：$E(X) = \int x f(x) dx$，$D(X) = E(X^2) - [E(X)]^2$
>
> ---
>
> **(1) 求 $E(X)$**
>
> $$E(X) = \int_0^1 x \cdot 3x^2 \, dx = \int_0^1 3x^3 \, dx = \left.\frac{3x^4}{4}\right|_0^1 = \frac{3}{4}$$
>
> ---
>
> **(2) 求 $E(X^2)$**
>
> $$E(X^2) = \int_0^1 x^2 \cdot 3x^2 \, dx = \int_0^1 3x^4 \, dx = \left.\frac{3x^5}{5}\right|_0^1 = \frac{3}{5}$$
>
> ---
>
> **(3) 求 $D(X)$**
>
> $$D(X) = E(X^2) - [E(X)]^2 = \frac{3}{5} - \left(\frac{3}{4}\right)^2 = \frac{3}{5} - \frac{9}{16} = \frac{48 - 45}{80} = \frac{3}{80}$$

**例3**：$X \sim B(10, 0.3)$，$Y = 2X + 1$，求 $E(Y), D(Y)$。

> [!example]- 答案（点击展开）
>
> **Step 1**：算 $X$ 的 $E$ 和 $D$
>
> $X \sim B(10, 0.3)$，套二项分布公式：
>
> $$E(X) = np = 10 \times 0.3 = 3, \quad D(X) = np(1-p) = 10 \times 0.3 \times 0.7 = 2.1$$
>
> ---
>
> **Step 2**：用线性变换公式（$Y = aX + b$）
>
> - $E(aX + b) = a E(X) + b$
> - $D(aX + b) = a^2 D(X)$
>
> $$E(Y) = 2 \times E(X) + 1 = 2 \times 3 + 1 = 7$$
>
> $$D(Y) = 2^2 \times D(X) = 4 \times 2.1 = 8.4$$

**例4**：$X,Y$ 独立，$D(X)=2$，$D(Y)=3$。求 $D(2X-3Y)$。

> [!example]- 答案（点击展开）
>
> $$D(2X-3Y) = 4D(X) + 9D(Y) = 8 + 27 = 35$$

### 5.3 协方差与相关系数

> [!NOTE] 5.3 这一节到底在学什么
> 5.1、5.2 都在研究**一个**随机变量的数字标签——$E(X)$ 描述中心，$D(X)$ 描述胖瘦。5.3 要研究**两个**变量之间的"关系标签"。
>
> 两个核心标签的分工：
>
> - $\text{Cov}(X,Y)$：**"两个变量是不是同涨同跌"的数字量度**（但**受单位影响**——比如身高从米改成厘米，Cov 数值会变 100 倍）
> - $\rho_{XY}$：**去掉单位后的纯版本**（永远在 $[-1, 1]$ 之间，不受量纲影响）
>
> 两个关键考点：
>
> 1. **计算**：$\text{Cov} = E(XY) - E(X)E(Y)$（其中 $E(XY)$ 要么有联合分布，要么靠**独立**条件 $E(XY)=E(X)E(Y)$）
> 2. **易错**：$E(XY) = E(X)E(Y)$ **只在独立时成立**
>
> （独立性判定的最大陷阱"不相关 ≠ 独立"见下方 [!WARNING]）
>
> 所以 5.3 可以理解为：**给"两个变量是否一起变化"贴标签——$\text{Cov}$ 反映方向和强度（含单位），$\rho$ 是去掉单位后的"纯版本"（永远是 $[-1, 1]$）。**

> [!WARNING] 5.3 第一陷阱：不相关 ≠ 独立
> **独立**是非常强的条件。独立 ⇒ Cov = 0，但 Cov = 0 **不一定**独立。
>
> **经典反例**：$X$ 取 $-1, 0, 1$ 各 $\frac{1}{3}$，$Y = X^2$。
>
> - 显然**不独立**（$Y=0$ 时 $X$ 必为 0，被 $Y$ 完全确定）
> - 但 $\text{Cov}(X, Y) = E(X \cdot X^2) - E(X)E(X^2) = E(X^3) - 0 = 0$
>
> **唯一例外**：**二维正态分布**中 $\rho = 0 \iff$ 独立。

> [!IMPORTANT] ⚡ 速算公式：看到 $Y = aX + b$ 必背（高频考点）
>
> | 量 | 公式 | 备注 |
> |----|------|------|
> | $\text{Cov}(X,\, aX+b)$ | $= \mathbf{a \cdot D(X)}$ | **常数 $b$ 没用**（自动消掉）|
> | $D(aX+b)$ | $= \mathbf{a^2 \cdot D(X)}$ | **常数 $b$ 没用**（自动消掉）|
> | $E(aX+b)$ | $= aE(X) + b$ | **这里 $b$ 才有用** |
>
> **口诀**：看到 $Y = aX + b$ 这种**线性关系**——**直接套上面三个公式**，不用展开 $E(XY)$ 算 Cov，不用硬算 $E(Y)$。
>
> **典型例题**：$X \sim N(0,1)$，$Y = 2X + 1$，求 Cov 和 $\rho$
> - $\text{Cov}(X, Y) = 2 \cdot D(X) = 2 \cdot 1 = 2$ ✓
> - $D(Y) = 2^2 \cdot D(X) = 4$ ✓
> - $\rho = \dfrac{2}{\sqrt{1} \cdot \sqrt{4}} = 1$ ✓（必然，因为 $Y = 2X+1$ 是严格线性关系）
>
> **记住**：$Y = aX + b$ 必然 $\rho = +1$（$a>0$）或 $\rho = -1$（$a<0$）。

#### 1. 协方差（衡量两变量同向变化）

**计算式（必用）**：
$$\text{Cov}(X,Y) = \underbrace{E(XY)}_{\text{乘积的期望}} - \underbrace{E(X)E(Y)}_{\text{各自期望的乘积}}$$

> 协方差 $Cov(X,Y) = E(XY) - E(X)E(Y)$，衡量 $X$ 与 $Y$ 的联合变动


**性质**：
- $\text{Cov}(X,Y) = \text{Cov}(Y,X)$
- $\text{Cov}(X,X) = D(X)$
- $\text{Cov}(aX, bY) = ab\,\text{Cov}(X,Y)$
- $\text{Cov}(X+Y, Z) = \text{Cov}(X,Z) + \text{Cov}(Y,Z)$
- $X,Y$ 独立 $\Rightarrow \text{Cov} = 0$

#### 2. 相关系数（标准化的协方差）

$$\rho_{XY} = \frac{\text{Cov}(X,Y)}{\sqrt{D(X)} \cdot \sqrt{D(Y)}}$$

> 相关系数 $\rho_{XY} = Cov(X,Y) / (\sigma_X \sigma_Y)$，协方差的标准化（消除量纲）


- $|\rho| \leq 1$
- $\rho = \pm 1 \iff Y = aX + b$（完全线性）
- $\rho > 0$：正相关；$\rho < 0$：负相关；$\rho = 0$：不相关

#### 3. 不相关 vs 独立

- 独立 $\Rightarrow$ 不相关（$\rho = 0$）
- 不相关 $\not\Rightarrow$ 独立
- 唯一特例：**二维正态**分布中 $\rho = 0 \iff$ 独立

**判别独立性的几种操作（按题目给的条件选）**：

| 关键操作 | 适用条件 | 结论强度 |
|---------|---------|---------|
| $p_{ij} = p_{i\cdot} p_{\cdot j}$ | 有联合分布（离散）| 独立 ↔ 不相关（**等价**）|
| $\rho = 0$ | 二维正态 | 独立 ↔ 不相关（**等价**）|
| $\rho = 0$ | 一般情况 | 仅"不相关"，**独立不知道** |
| $E(XY) = E(X)E(Y)$ | 一般情况 | 仅"不相关"，**独立不知道** |

**具体例子**：

> [!example]- 例 1（有联合分布 + 独立）
>
> | $X \setminus Y$ | 0 | 1 |
> |---|---|---|
> | 0 | 0.15 | 0.35 |
> | 1 | 0.15 | 0.35 |
>
> - 边际：$P(X=0)=0.5$，$P(X=1)=0.5$，$P(Y=0)=0.3$，$P(Y=1)=0.7$
> - 验证 $p_{00} = 0.15 = 0.5 \times 0.3$ ✓
> - 四个格子全部满足 $p_{ij} = p_{i\cdot} p_{\cdot j}$ → **独立** ✓

> [!example]- 例 2（有联合分布 + 不独立）
>
> | $X \setminus Y$ | 0 | 1 |
> |---|---|---|
> | 0 | 0.3 | 0.2 |
> | 1 | 0.1 | 0.4 |
>
> - 边际：$P(X=0)=0.5$，$P(Y=0)=0.4$
> - 验证 $p_{00} = 0.3 \ne 0.5 \times 0.4 = 0.2$ ✗
> - **不独立**（一个反例就够了）

> [!example]- 例 3（一般情况，$\rho = 0$ 不一定独立）
>
> $X$ 取 $-1, 0, 1$ 各 $\frac{1}{3}$，$Y = X^2$。
> - $\text{Cov}(X, Y) = E(X \cdot X^2) - E(X)E(X^2) = E(X^3) - 0 = 0$ → $\rho = 0$
> - 但 $Y = 0$ 时 $X$ 必为 $0$ → **不独立** ✗
> - 所以 $\rho = 0$ **不能**推出独立

> [!example]- 例 4（二维正态，$\rho = 0$ 等价于独立）
>
> $(X, Y)$ 服从二维正态，$\rho = 0$。
> - → **独立** ✓
> - 这是表中"$\rho = 0$ ↔ 独立"唯一成立的场景

#### 4. $E(XY)$ 的求法

- **独立**：$E(XY) = E(X)E(Y)$
- **不独立**（按定义）：$E(XY) = \sum_i \sum_j x_i y_j \, p_{ij}$（二维离散型）

#### 经典例题

**例1**：联合分布律如下，求 $\text{Cov}$ 和 $\rho$。

| $X \setminus Y$ | 0 | 1 |
|---|---|---|
| 0 | 0.3 | 0.2 |
| 1 | 0.1 | 0.4 |

> [!example]- 答案（点击展开）
>
> **公式**：$\text{Cov}(X,Y) = E(XY) - E(X)E(Y)$，$\rho = \dfrac{\text{Cov}(X,Y)}{\sqrt{D(X)}\sqrt{D(Y)}}$
>
> ---
>
> **Step 1**：算 $E(X)$（按行求和得 $X$ 的边缘）
>
> $P(X=0) = 0.3 + 0.2 = 0.5$，$P(X=1) = 0.1 + 0.4 = 0.5$
>
> $$E(X) = 0 \times 0.5 + 1 \times 0.5 = 0.5$$
>
> **Step 2**：算 $E(Y)$（按列求和得 $Y$ 的边缘）
>
> $P(Y=0) = 0.3 + 0.1 = 0.4$，$P(Y=1) = 0.2 + 0.4 = 0.6$
>
> $$E(Y) = 0 \times 0.4 + 1 \times 0.6 = 0.6$$
>
> **Step 3**：算 $E(XY)$（每个 $x_i y_j$ 乘对应概率再加总）
>
> $$E(XY) = 0 \cdot 0 \cdot 0.3 + 0 \cdot 1 \cdot 0.2 + 1 \cdot 0 \cdot 0.1 + 1 \cdot 1 \cdot 0.4 = 0.4$$
>
> **Step 4**：算 $\text{Cov}$
>
> $$\text{Cov}(X,Y) = E(XY) - E(X)E(Y) = 0.4 - 0.5 \times 0.6 = 0.1$$
>
> **Step 5**：算 $D(X), D(Y)$
>
> $$D(X) = E(X^2) - [E(X)]^2 = (0^2 \cdot 0.5 + 1^2 \cdot 0.5) - 0.25 = 0.25$$
> $$D(Y) = E(Y^2) - [E(Y)]^2 = (0^2 \cdot 0.4 + 1^2 \cdot 0.6) - 0.36 = 0.24$$
>
> **Step 6**：算 $\rho$
>
> $$\rho = \frac{0.1}{\sqrt{0.25 \times 0.24}} = \frac{0.1}{\sqrt{0.06}} \approx 0.408$$

**例2**：$X \sim N(0,1)$，$Y = 2X + 1$。求 $\text{Cov}$ 和 $\rho$。

> [!example]- 答案（点击展开）
>
> **Step 1**：算 $\text{Cov}(X, 2X+1)$
>
> 用公式：$\text{Cov}(X, aX+b) = a \cdot D(X)$
>
> $$\text{Cov}(X, 2X+1) = 2 \cdot D(X) = 2 \cdot 1 = 2$$
>
> **Step 2**：算 $\rho$
>
> $\rho = \dfrac{\text{Cov}(X,Y)}{\sqrt{D(X)} \cdot \sqrt{D(Y)}}$
>
> - $D(X) = 1$
> - $D(Y) = 2^2 D(X) = 4$
>
> $$\rho = \frac{2}{\sqrt{1} \cdot \sqrt{4}} = \frac{2}{2} = 1$$
>
> **结论**：$\rho = 1$ 表示 $Y$ 与 $X$ **完全正线性相关**（因为 $Y = 2X+1$ 是严格线性关系）

#### 易错点

| 易错点 | 正确 |
|--------|------|
| 不相关 = 独立 | 不相关 ≠ 独立 |
| $E(XY) = E(X)E(Y)$ 恒成立 | 仅独立时 |
| $\text{Cov}$ 忘负号 | $\text{Cov} = E(XY) - E(X)E(Y)$ |
| $\rho$ 分母忘开根号 | $\sqrt{D(X)}\sqrt{D(Y)}$ |

## 六、参数估计

> 考纲对应：**考点 3**（简单连续型随机变量求矩估计量）
> 参考页码：P132 例 6.2、P142 第 7 题

> [!NOTE] 第 6 章到底在学什么
> 前 5 章都在研究"已知总体分布求统计量"——总体分布是给定的。第 6 章反过来了：**总体分布的形式已知，但参数未知，要用样本来反推参数**。
>
> 核心思想只有一句话：**用样本矩代替总体矩**。
>
> 考试题本质上就是：
>
> **给一个含未知参数 $\theta$ 的总体分布，给一组样本观测值（或样本均值 $\bar{X}$、二阶中心矩 $B_2$），列方程 $E(X)=\bar{X}$（或更多阶），解出 $\hat\theta$。**
>
> 所以第 6 章可以理解为：**"已知样本来猜总体参数"——一句话口诀"用样本矩代总体矩"，配上解方程的套路。**

### 6.1 矩的定义

| 名称 | 总体矩 | 样本矩 |
|------|--------|--------|
| $k$ 阶**原点矩** | $E(X^k)$ | $A_k = \dfrac{1}{n}\sum_{i=1}^n X_i^k$ |
| $k$ 阶**中心矩** | $E[(X-E(X))^k]$ | $B_k = \dfrac{1}{n}\sum_{i=1}^n (X_i - \bar{X})^k$ |

### 6.2 矩估计的核心思想

> **用样本矩估计总体矩**

- 总体期望（一阶原点矩）→ 用**一阶样本原点矩**（即样本均值）估计
- 总体方差（二阶中心矩）→ 用**二阶样本中心矩** $B_2$ 估计

### 6.3 矩估计公式（必背）

**（1）总体期望的矩估计**：
$$\hat{\mu} = \underbrace{E(X)}_{\text{总体期望}} = \underbrace{\bar{X}}_{\text{样本均值}} = \underbrace{\frac{1}{n}\sum_{i=1}^n X_i}_{\text{样本平均}}$$

> $\hat{\mu}$ 是总体期望的矩估计量：$\hat{\mu} = \bar{X} = \frac{1}{n} \sum X_i$


**（2）总体方差的矩估计**：
$$\hat{\sigma}^2 = \underbrace{B_2}_{\text{样本二阶中心矩}} = \underbrace{\frac{1}{n}\sum_{i=1}^n (X_i - \bar{X})^2}_{\text{偏差平方的平均}}$$

> $\hat{\sigma}^2$ 是总体方差的矩估计量：$\hat{\sigma}^2 = B_2 = \frac{1}{n} \sum (X_i - \bar{X})^2$


>  注意：$\hat{\sigma}^2 = B_2$（除以 $n$），不是除以 $n-1$。除以 $n-1$ 的是**无偏估计** $S^2$，不在本考点范围内。

### 6.4 解题套路（4 步）

1. 写出总体的 $E(X)$（含未知参数 $\theta$）
2. 列等式：$E(X) = \bar{X} = \dfrac{1}{n}\sum X_i$
3. 解出 $\hat{\theta}$
4. 写出估计量表达式（用 $\bar{X}$ 表示）

### 6.5 经典例题

**例1**：设 $X \sim$ 指数分布 $E(\lambda)$，求 $\lambda$ 的矩估计量。

> [!example]- 答案（点击展开）
>
> **Step 1**：写出 $E(X)$
>
> 指数分布的期望：$E(X) = \dfrac{1}{\lambda}$
>
> **Step 2**：列等式 $E(X) = \bar{X}$，解出 $\lambda$
>
> $$\frac{1}{\lambda} = \bar{X}$$
>
> $$\hat{\lambda} = \frac{1}{\bar{X}}$$

**例2**：设 $X$ 的密度为 $f(x;\theta) = \begin{cases} \dfrac{x}{\theta^2} e^{-x/\theta} & x > 0 \\ 0 & \text{其他} \end{cases}$（$\theta>0$），求 $\theta$ 的矩估计量。

> [!example]- 答案（点击展开）
>
> **Step 1**：算 $E(X)$
>
> $$E(X) = \int_0^\infty x \cdot \frac{x}{\theta^2} e^{-x/\theta} \, dx = \int_0^\infty \frac{x^2}{\theta^2} e^{-x/\theta} \, dx$$
>
> 这是**伽马分布** $Gamma(2, \theta)$，均值为 $2\theta$（不需要详细算分步积分）
>
> $$E(X) = 2\theta$$
>
> **Step 2**：列等式 $E(X) = \bar{X}$，解出 $\theta$
>
> $$2\theta = \bar{X}$$
>
> $$\hat{\theta} = \frac{\bar{X}}{2}$$

**例3**：设 $X \sim U(0, \theta)$，求 $\theta$ 的矩估计量。

> [!example]- 答案（点击展开）
>
> **Step 1**：算 $E(X)$
>
> 均匀分布 $U(0, \theta)$ 的期望：$E(X) = \dfrac{0 + \theta}{2} = \dfrac{\theta}{2}$
>
> **Step 2**：列等式 $E(X) = \bar{X}$，解出 $\theta$
>
> $$\frac{\theta}{2} = \bar{X}$$
>
> $$\hat{\theta} = 2\bar{X}$$

### 6.6 易错点

| 易错点 | 正确理解 |
|--------|----------|
| 矩估计用 $\bar{X}$ 代替 $E(X^2)$ | 应该代替 $E(X)$（一阶原点矩），不是 $E(X^2)$ |
| $\hat{\sigma}^2 = S^2$（除以 $n-1$） | 本考点用 $B_2$（除以 $n$）。$S^2$ 是无偏估计 |
| 矩估计公式忘了平方 | $B_2 = \frac{1}{n}\sum (X_i - \bar{X})^2$，要平方 |

## 附录 A：速记速查

### 通用方法

- 标准解题：连等代入（公式 = 数据 = 结果），答案用**分数**别用小数
- 离散看"点"（P(X=k)），连续看"段"（P(a<X<b)）
- 例题答案默认折叠，做题时收起，看答案时展开

### 概率公式速记

- 减法：$P(A-B) = P(A) - P(AB)$（A 去掉 A∩B 的部分）
- 条件概率：**缩小样本空间**，$P(A|B) = P(AB)/P(B)$
- 乘法公式：$P(AB) = P(B)·P(A|B) = P(A)·P(B|A)$
- 独立：B 不改变 A 的概率，即 $P(A|B) = P(A)$
- 互斥：$P(AB) = 0$；独立：$P(AB) = P(A)P(B)$
- 全概率：**因的比例 × 该因下 A 发生，所有因加起来**
- 贝叶斯：**一个加数 / 总和**

### 伯努利/二项速记

- 伯努利 = 抛硬币；n 重伯努利 → 二项分布 $X \sim B(n, p)$
- 公式 k **必须对齐**（C 的 k = $p^k$ 的指数 = $(1-p)^{n-k}$ 的指数）
- 选正面或反面等价，只要 k 对齐
- "至少 K 次" 用对立事件（$1 - P(\text{少于 K})$）
- 二项/泊松：n 有限（抛 n 次）→ 二项；n 没给（稀有计数）→ 泊松

### 数字特征速记

- E(X) = **加权平均**
- D(X) = **平方均值 − 均值平方** = $E(X^2) - [E(X)]^2$
- 泊松特征：$E(X) = D(X) = \lambda$
- 正态特征：$E(X) = \mu$，$D(X) = \sigma^2$

### F(x) 与 f(x) 速记

| | F(x) | f(x) |
|---|---|---|
| 关系 | 累积概率 | 概率密度 |
| 互为 | f 的原函数 | F 的导函数 |
| 区间 | $F(b) - F(a)$ | $\int_a^b f(x)dx$ |

### 矩估计速记

- 核心：$E(X) = \bar{X}$，$E(X^2) = A_2$（用样本矩代总体矩）
- 几个未知参数 = 几阶矩 = 几个方程

## 附录 C：高频易错点汇总

#### 概率运算

| 错误 | 正确 |
|------|------|
| $P(A-B) = P(A) - P(B)$ | $P(A-B) = P(A) - P(AB)$ |
| 贝叶斯忘分母 | 分母 = 全概率 |

#### 独立性

| 错误 | 正确 |
|------|------|
| $D(X \pm Y) = D(X)+D(Y)$ 总成立 | 仅当 $X,Y$ 独立 |
| $E(XY) = E(X)E(Y)$ 总成立 | 仅当 $X,Y$ 独立 |
| 不相关 = 独立 | 不相关 ≠ 独立 |
| 互不相容 → 独立 | 互斥且 $P>0$ → 不独立 |

#### 连续型

| 错误 | 正确 |
|------|------|
| $P(X=a) > 0$ | 连续型单点概率为 0 |
| 区间开闭影响概率 | 连续型中完全相同 |
| $f(x)$ 是概率 | $f(x)dx$ 才是概率微元 |

#### 正态分布

| 错误 | 正确 |
|------|------|
| $\Phi(-x) = -\Phi(x)$ | $\Phi(-x) = 1 - \Phi(x)$ |
| 标准化除以 $\sigma^2$ | 除以 $\sigma$（标准差） |

#### 方差与系数

| 错误 | 正确 |
|------|------|
| $D(cX) = cD(X)$ | $D(cX) = c^2 D(X)$ |
| $D(2X) = 2D(X)$ | $D(2X) = 4D(X)$ |
